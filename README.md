# Kellanova (kellanov)
Kellanova was a Fortune 500 snack foods company spun off from Kellogg Company in October 2023, known for brands like Pringles, Cheez-It, and Pop-Tarts. In December 2025, Mars Inc. completed its acquisition of Kellanova for nearly $30 billion. Kellanova did not offer a public API developer portal or externally documented REST APIs for third-party developer integration.
