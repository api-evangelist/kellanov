# Kellanov (kellanov)

Kellanov is a tracking alias profile for Kellanova, the global snacks-led consumer packaged foods company formed in 2023 from the spin-off of the Kellogg Company's snacks, international cereal and noodles, and North American frozen breakfast businesses. Brands include Pringles, Cheez-It, Pop-Tarts, Eggo, RXBAR, MorningStar Farms, and Special K outside North America. The canonical profile is maintained at api-evangelist/kellanova.
